The PatchLab documentation does not intentionally use analytics, advertising, accounts,
fingerprinting, or lesson-progress cookies. Interactive DSP labs run in the page and do
not send their control values to PatchLab servers.

The hosting and DNS providers may process ordinary request and security logs. External
references and GitHub links are governed by their destinations. If the site later adds
analytics, embedded media, feedback widgets, accounts, or server-side lesson state, the
privacy policy must be revised before those features are enabled.

Do not place credentials, personal paths, private audio, or confidential projects in a
public documentation issue.
