## A patch is a signal system

<div class="pl-signal-chain">
	<div><strong>Source</strong><span>Audio input, oscillator, sampler</span></div>
	<div><strong>Shape</strong><span>Filter, dynamics, distortion</span></div>
	<div><strong>Move</strong><span>LFO, envelope, follower</span></div>
	<div><strong>Observe</strong><span>Meter, scope, spectrum</span></div>
	<div><strong>Deliver</strong><span>Perform, save, export</span></div>
</div>

The documentation follows the same modular idea as PatchLab. Read one concept in
isolation, follow its links into a working guide, or open the exact block reference.
You should never need to read the site in order.
