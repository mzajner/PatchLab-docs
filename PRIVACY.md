# Privacy policy

The PatchLab documentation is a static site. It does not intentionally set analytics,
advertising, account, fingerprinting, or lesson-progress cookies, and it does not send
interactive-lab values to a server.

The hosting and DNS providers may process ordinary request metadata such as IP address,
user agent, requested URL, timestamp, and security logs under their own policies.
Following an external reference or GitHub link leaves the PatchLab documentation site
and is governed by that destination.

If analytics, embedded media, feedback widgets, accounts, or server-side lesson state
are added later, this policy and the site must be updated before those features are
enabled. Security reports should use the private process in `SECURITY.md`; do not place
personal or confidential information in a public issue.
